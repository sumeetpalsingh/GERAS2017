Classification of zebrafish beta-cells

This is an R Markdown document (http://rmarkdown.rstudio.com) detailing the steps for building the age classifier for zebrafish beta-cells.

Zebrafish beta-cells were collected from seven stages: 1, 3, 4, 6, 10, 12, 14 month post fertilization (mpf - time since birth). From each stage, individual beta-cells were sorted into wells of 96-well plate and sequenced. The raw data from single-cell sequencing was trimmed, mapped, quality controled and normalized to TPM (transcripts per million) values. Details of each step are provided in accompanying files (refer to ReadMe.txt for file information.)

The document starts with compiled TPM values to build a classifier using Deep Neural Network (DNN) that separates individual cells into the following three age groups:

1 mpf: Juvenile

3, 4, 6 mpf: Young Adult

10, 12, 14 mpf: Old Adult

The DNN is build using TensorFlow library.

Cross-validation for setting hyperparameters

## To obtain the regularization hyperparameters, we will perform k-fold cross-validation
## We will perfrom 5-fold cross validation repeated 3 times
## The cross validation will be performed over a grid of dropout constant and L2 regularization constant
## droput values will range from 0.4 - 0.9 (6 values)
## Regularization constant values will range from 0.4 - 1.6 in steps of 0.2 (7 values)
nFolds = 5
crossValidation.repeat = 3
dropout.test = seq.int(0.4, 0.9, 0.1)
regularization.test = seq.int (0.4, 1.6, 0.2)
## To save the accuracy of the models
accuracy.models <- matrix(NA, nrow = length(dropout.test), ncol = length(regularization.test))
## We will generate specific seed for each model to enable reproducibility
set.seed(177)
repeat.seeds <- sample(1:1000, crossValidation.repeat)
models.seeds <- array(sample(1:1000, length(accuracy.models)*crossValidation.repeat),
                      dim = c(length(dropout.test), length(regularization.test), crossValidation.repeat))

for (dropout.index in 1:length(dropout.test)){
  for(regularization.index in 1:length(regularization.test)){
    accuracy.repeat = 0
    for(crossValidation.index in 1:crossValidation.repeat){
      # print(models.seeds[dropout.index, regularization.index])
      set.seed(models.seeds[dropout.index, regularization.index, crossValidation.index])
      training.Folds = createFolds(1:ncol(train.log), k = nFolds, list = TRUE, returnTrain = TRUE)
      for(nFolds.index in 1:nFolds){
        train.fold = train.log[,training.Folds[[nFolds.index]]]
        train.cond.fold = train.cond[training.Folds[[nFolds.index]]]
        test.fold = train.log[,-training.Folds[[nFolds.index]]]
        test.cond.fold = train.cond[-training.Folds[[nFolds.index]]]
        
        parameters.weights.fold = model_dropout(X_train = train.fold, Y_train = train.cond.fold,
                                                num_epochs = 250, learning_rate = 0.001,
                                                regulatization.constant = regularization.test[regularization.index],
                                                dropout.constant = dropout.test[dropout.index],
                                                print_cost = F)
        accuracy.current = prediction_accuracy(test.fold, test.cond.fold, parameters.weights.fold)
        accuracy.repeat = accuracy.repeat + accuracy.current
      }
    }
    accuracy.models[dropout.index, regularization.index] = accuracy.repeat/(crossValidation.repeat * nFolds)
    print(paste("Accuracy for dropout", dropout.test[dropout.index], 
                ", regularization", regularization.test[regularization.index],
                "is:", accuracy.models[dropout.index, regularization.index]))
  }
}
